    • Matrix: Element Click here

    Matrix is an open protocol for instant messaging and file sharing with voice and video support.

    It is a decentralized client-server protocol with server-to-server messaging.

    An API (program interaction interface) in JSON format was also developed.

    The protocol provides the ability to exchange messages with other protocols through bridges.

    Bridges are programs that work in conjunction with a server and forward messages between different networks. Bridges currently exist for:

    1. Telegram

    2. Discord

    3. Gitter

    4. Slack/Mattermost

    5. IRC

    6. WhatsApp

    7. XMPP

    8. Facebook Messenger

    9. Signal

    10. Skype

    Status is a secure messenger with E2EE support, a built-in ETH wallet and support for DApps (decentralized applications).

    It uses the Waku protocol (a fork of Whisper) for P2P communication.

    • Briar: Click here

    Briar is a messaging app designed for activists, journalists, and anyone else who needs a safe, easy and robust way to communicate.

    Unlike traditional messaging apps, Briar doesn’t rely on a central server - messages are synchronized directly between the users’ devices.

    If the internet’s down, Briar can sync via Bluetooth or Wi-Fi, keeping the information flowing in a crisis.

    If the internet’s up, Briar can sync via the Tor network, protecting users and their relationships from surveillance.

    - Doesn't support image's

    - Doesn't support GIF

    - Doesn't support Emodji

    - Doesn't support Video\audio calls

    - Doesn't support Voice messages
